Установка:а. Создайте БД для личного кабинета с любым названием ( я назвал mylk ).  б. Из папки /sql залейте файл vote_base в БД личного кабинета ( в моём случае mylk ).  в. Из папки /sql залейте файл realm в бд авторизации. realmd для мангос, auth для тринити.  г. Откройте config.php в папке /trunk/include и заполните  д. Откройте config.php в папке /trunk/lagranj и заполните  е. Перейдите к спойлеру "Управление сайтом"  Управление сайтом ( файлы для редактирования ): Новости - trunk/lagranj/news.php  Регистрация - /reg.php  Телепорт - /tele.php  Всё что внутри тэга <head> - /trunk/lagranj/header.php  Коопирайт - /trunk/lagranj/footer.php ( lagranj и iUnro не снимать ).  Вход в личный кабинет, версия сервера - /trunk/lagranj/top.php  Название реалма, рейты, реалмлист, меню, группа вконтакте - /trunk/lagranj/menu.php  Рекламный блок - /trunk/lagranj/rek.php  В личном кабинете, изменить "Ваш сервер" - /trunk/index.php  Меню личного кабинета - /trunk/lagranj/m.php  Дизайн модулей, соглашение и т.д - /trunk/module/   Смена логотипа: Папка trunk/img/layuot/ картинка ks_logo_transparent.png - Лого сайта.  PSD - ks_logo_transparent.psd - в той же папке.  Если нету навыков или возможности сменить лого, удалите ks_logo_transparent.png и его просто не будет.  Личный кабинет:Для добавления топа в панель голосования выполните следующий запрос в БД личного кабинета: INSERT INTO `game_top` (`name`,`link`,`cost`) VALUES ('Имя','http://ссылка/на/ваш/сервер/в/топе','Количество_бонусов'); Для добавления вещи в магазин вашего сервера выполните следующий запрос в БД личного кабинета: INSERT INTO `item_list` VALUES ('ИД_ВЕЩИ','ИМЯ_ВЕЩИ','ЦЕНА_В_БОНУСАХ'); Конечно всё можно сделать в ручную через навикат.  Таблица с топами - game_top  Таблица с вещами - item_list  Цена бонусов за смену рассы, класса, перенос и т.д указываете в /trunk/include/config.php
